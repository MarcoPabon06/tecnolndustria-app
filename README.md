# tecnolndustria-app

# 🏭 Tecnolndustria S.A.S - Panel de Control Físico

Una solución tecnológica desarrollada para optimizar los procesos de producción y distribución de materiales industriales de Tecnolndustria S.A.S. mediante el análisis y control de variables físicas en tiempo real.

## 📋 Descripción del Proyecto

La empresa Tecnolndustria S.A.S. enfrentaba problemas críticos en el control de variables físicas, lo que ocasionaba fallas en los procesos, desperdicio de recursos y altos costos operativos. 

Este proyecto es una *Single Page Application* (SPA) que proporciona a los operarios e ingenieros un panel de control unificado para realizar cálculos, simulaciones y análisis rápidos de propiedades físicas esenciales, facilitando la toma de decisiones y mejorando la eficiencia energética de la maquinaria.

## ✨ Características Principales

El software incluye calculadoras interactivas basadas en principios y leyes de la física para las siguientes variables:

* **🔥 Calor (Energía Térmica):** Analiza procesos de calentamiento/enfriamiento ($Q=m \cdot c \cdot \Delta T$).
* **🌡️ Temperatura:** Conversión precisa entre grados Celsius, Fahrenheit y Kelvin.
* **⚙️ Presión:** Cálculo de fuerza aplicada sobre un área específica para sistemas hidráulicos ($P=F/A$).
* **💧 Caudal Volumétrico y por Área:** Monitoreo del flujo en tuberías y procesos productivos.
* **🏗️ Energía Potencial:** Cálculos para sistemas de elevación y almacenamiento ($Ep=m \cdot g \cdot h$).
* **⚡ Eficiencia Energética:** Medición del rendimiento del sistema para reducción de costos ($\eta=(E.útil/E.total) \times 100$).

## 🛠️ Tecnologías Utilizadas

Este proyecto fue construido utilizando tecnologías web estándar, garantizando un despliegue rápido y una alta compatibilidad sin necesidad de servidores complejos:

* **HTML5:** Estructura semántica de la aplicación.
* **CSS3:** Estilos personalizados, diseño responsivo (adaptable a móviles y escritorio) y variables CSS para el manejo de temas.
* **JavaScript (Vanilla):** Lógica de navegación del DOM y algoritmos de cálculo físico en tiempo real.
